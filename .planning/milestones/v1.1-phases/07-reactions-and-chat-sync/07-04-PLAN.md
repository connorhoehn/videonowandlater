```typescript
export function useSynchronizedChat(
  allMessages: ChatMessage[],
  currentSyncTime: number
): ChatMessage[] {
  return useMemo(() => {
    if (currentSyncTime === 0) return [];
    return allMessages.filter(
      msg => msg.sessionRelativeTime !== undefined &&
             msg.sessionRelativeTime <= currentSyncTime
    );
  }, [allMessages, currentSyncTime]);
}
```

- reactions: Reaction[] (all reactions for session)
- currentTime: number (syncTime from player)
- duration: number (video duration in ms)

Component logic:
1. Fetch all reactions on mount via GET /sessions/:sessionId/reactions
2. Aggregate reactions into 5-second buckets using useMemo:
   - Create Map<bucketNumber, { emojis: string[], count: number }>
   - bucket = Math.floor(sessionRelativeTime / 5000)
   - Accumulate count per bucket, collect unique emoji types
3. Render timeline markers:
   - Position each bucket marker at (bucket * 5000 / duration) * 100%
   - Show reaction count badge
   - Show emoji icons for unique types in bucket
   - Highlight marker if currentTime >= bucket start
4. Style as horizontal bar below video scrubber (absolute positioning)

Use Tailwind for styling (small circles/badges, emoji icons, highlighted state).
