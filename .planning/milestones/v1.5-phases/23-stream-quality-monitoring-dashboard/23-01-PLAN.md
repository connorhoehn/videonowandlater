Create `web/src/domain/metrics.ts`:

1. Define `StreamMetrics` interface:
   - `timestamp: number` (Date.now() when sampled)
   - `bitrate: number` (bytes sent from RTCOutboundRtpStreamStats)
   - `framesPerSecond: number`
   - `resolution: { width: number; height: number }`
   - `networkType: string` (default 'unknown')
   - `qualityLimitation: string` (default 'none')
   - `jitter?: number` (optional)
   - `packetsLost?: number` (optional)

2. Define `HealthScoreResult` interface:
   - `score: number` (0-100)
   - `bitrateHealth: number` (0-100)
   - `fpsHealth: number` (0-100)
   - `warning: 'none' | 'bitrate-drop' | 'fps-drop' | 'both'`

3. Implement `calculateHealthScore(inputs: HealthScoreInputs): HealthScoreResult`:
   - `HealthScoreInputs`: currentBitrate, targetBitrate, currentFps, targetFps, recentBitrates[], recentFrameRates[]
   - Bitrate health calculation:
     * Compute average bitrate from recentBitrates array
     * Compute bitrateDifference = abs(avg - target) / target
     * Compute bitrateVariance = stdDev(recentBitrates) / avg (jitter penalty)
     * bitrateHealth = max(0, 100 - bitrateDifference * 40 - bitrateVariance * 60)
   - FPS health calculation:
     * fpsOnTarget = percentage of samples >= targetFps * 0.95
     * fpsVariance = stdDev(recentFrameRates) / targetFps
     * fpsHealth = max(0, fpsOnTarget * 80 - fpsVariance * 20)
   - Combined score: `bitrateHealth * 0.6 + fpsHealth * 0.4`
   - Warning detection:
     * bitrateDropped = avgBitrate < targetBitrate * 0.7 (>30% below)
     * fpsCritical = fpsOnTarget < 0.5
     * Set warning based on both flags

4. Implement helper `stdDev(samples: number[]): number`:
   - Standard deviation formula: sqrt(variance)
   - Variance = sum((x - mean)^2) / length

5. Export all types and functions

Create `web/src/features/broadcast/useStreamMetrics.ts`:

1. Import dependencies:
   - `useState`, `useEffect`, `useRef` from 'react'
   - `StreamMetrics`, `HealthScoreResult`, `calculateHealthScore` from '../../domain/metrics'

2. Define `extractStreamStats(broadcastClient: any): Promise<StreamMetrics | null>`:
   - Guard: if (!broadcastClient?.peerConnection) return null
   - Call `await broadcastClient.peerConnection.getStats()`
   - Iterate reports, find `report.type === 'outbound-rtp' && report.mediaType === 'video'`
   - If not found, return null
   - Return StreamMetrics object:
     * timestamp: Date.now()
     * bitrate: outboundStats.bytesSent || 0
     * framesPerSecond: outboundStats.framesPerSecond || 0
     * resolution: { width: frameWidth || 0, height: frameHeight || 0 }
     * networkType: outboundStats.networkType || 'unknown'
     * qualityLimitation: outboundStats.qualityLimitation || 'none'
     * jitter: outboundStats.jitter
     * packetsLost: outboundStats.packetsLost

3. Export `useStreamMetrics(broadcastClient: any, isLive: boolean)`:
   - State: `const [metrics, setMetrics] = useState<StreamMetrics | null>(null)`
   - State: `const [healthScore, setHealthScore] = useState<HealthScoreResult | null>(null)`
   - Ref: `const samplesRef = useRef<{ bitrates: number[]; fpss: number[] }>({ bitrates: [], fpss: [] })`
   - Ref: `const pollIntervalRef = useRef<NodeJS.Timeout | null>(null)`
   - Ref: `const previousSampleRef = useRef<{ bytesSent: number; timestamp: number } | null>(null)`

4. useEffect for polling (deps: [isLive, broadcastClient]):
   - Guard: if (!isLive || !broadcastClient) return
   - Define async `poll()` function:
     * Try/catch wrapper
     * Call extractStreamStats(broadcastClient)
     * If null, return early
     * Calculate instantaneous bitrate:
       - If previousSampleRef.current exists:
         * bytesDelta = newMetrics.bitrate - prev.bytesSent
         * timeDelta = (newMetrics.timestamp - prev.timestamp) / 1000 (seconds)
         * instantaneousBitrate = (bytesDelta / timeDelta) * 8 (convert to bits/sec)
       - Else: use newMetrics.bitrate directly (first sample)
       - Store current sample: previousSampleRef.current = { bytesSent: newMetrics.bitrate, timestamp: newMetrics.timestamp }
     * Update samples rolling window:
       - Push instantaneousBitrate to samplesRef.current.bitrates
       - Push framesPerSecond to samplesRef.current.fpss
       - If length > 60, shift oldest sample
     * Calculate health score (only if we have at least 3 samples):
       - Target bitrate: 2500 * 1000 * 8 (2500 kbps in bits/sec)
       - Target FPS: 30 (BASIC_FULL_HD_LANDSCAPE default)
       - Call calculateHealthScore with inputs
       - setHealthScore(result)
     * setMetrics(newMetrics)
   - Call poll() immediately (first poll on mount)
   - Set interval: `pollIntervalRef.current = setInterval(poll, 5000)`
   - Cleanup: `if (pollIntervalRef.current) clearInterval(pollIntervalRef.current)`

5. Return `{ metrics, healthScore }`
