5. Prepare chart data (only last 12 samples for visibility):
   ```typescript
   const chartData = useMemo(() => {
     if (!historicalData || historicalData.length === 0) return [];
     // Take last 12 samples (1 minute at 5s intervals)
     return historicalData.slice(-12).map(d => ({
       time: d.time.split(':').slice(1).join(':'), // MM:SS format
       bitrate: Math.round(d.bitrate / 1000), // Convert to kbps
       fps: Math.round(d.fps),
     }));
   }, [historicalData]);
   ```

7. Helper component `MetricRow`:
   ```typescript
   function MetricRow({ label, value, alert }: { label: string; value: string; alert?: boolean }) {
     return (
       <div className={`flex justify-between text-xs ${alert ? 'text-yellow-400' : 'text-gray-300'}`}>
         <span>{label}</span>
         <span className="font-mono">{value}</span>
       </div>
     );
   }
   ```

8. Helper function `formatBitrate(bytes: number): string`:
   - Note: `bytes` from RTCOutboundRtpStreamStats.bytesSent is cumulative
   - useStreamMetrics already converted to instantaneous bitrate in bits/sec
   - Convert to kbps: `const kbps = bytes / 1000`
   - Return: `${Math.round(kbps)} kbps`
