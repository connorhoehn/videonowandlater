       ```
         (s.convertStatus === 'processing' || s.convertStatus === 'pending')
       );
       ```

       - Stores interval ID in `pollIntervalRef.current`
       - Returns cleanup function that clears the interval

    5. Reset pollInterval back to 15000 when sessions change from all-terminal to having non-terminal (so new processing sessions get fast polling). Add this logic: when `hasNonTerminalSessions` transitions from false to true, reset `setPollInterval(15000)`.
