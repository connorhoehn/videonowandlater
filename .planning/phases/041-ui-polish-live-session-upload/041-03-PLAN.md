    2. Add terminal state helper function (inside the module, before VideoPage component):

       ```typescript
       function isUploadTerminal(session: UploadSession): boolean {
         const anyFailed = [session.convertStatus, session.transcriptStatus, session.aiSummaryStatus, session.recordingStatus]
           .some(s => s === 'failed');
         return anyFailed || session.aiSummaryStatus === 'available';
       }
       ```

    3. In `VideoPage` component, add state/ref after existing state declarations (after `const [showInfoPanel, setShowInfoPanel]`):
       ```typescript

       const pollIntervalRef = useRef<ReturnType<typeof setInterval> | null>(null);
       ```

    4. Add seekVideo callback after the `useHlsPlayer` call:
       ```typescript
       const seekVideo = (timeMs: number) => {
         if (videoRef.current) {
           videoRef.current.currentTime = timeMs / 1000;
         }
       };
       ```

    5. Add polling useEffect after the existing session fetch useEffect (before the `useHlsPlayer` call). The polling effect must:
       - Clear any existing interval at effect start
       - Return early if no session or session is terminal
       - Otherwise start interval that: re-fetches session (same fetch pattern as initial fetch), calls `setSession(data)`, calls `setPollInterval(prev => Math.min(prev * 2, 60000))`
       - Return cleanup that clears the interval

       Full polling useEffect:
       ```typescript
       useEffect(() => {
         if (pollIntervalRef.current) {
           clearInterval(pollIntervalRef.current);
           pollIntervalRef.current = null;
         }
         if (!session || isUploadTerminal(session) || !authToken || !sessionId) return;
         const config = getConfig();
         const apiBaseUrl = config?.apiUrl || 'http://localhost:3000/api';
         const intervalId = setInterval(async () => {
           try {
             const response = await fetch(`${apiBaseUrl}/sessions/${sessionId}`, {
               headers: { 'Authorization': `Bearer ${authToken}` },
             });
             if (response.ok) {
               const data: UploadSession = await response.json();
               setSession(data);
             }
           } catch {
             // silent — polling errors should not surface as errors
           }
           setPollInterval(prev => Math.min(prev * 2, 60000));
         }, pollInterval);
         pollIntervalRef.current = intervalId;
         return () => {
           clearInterval(intervalId);
           pollIntervalRef.current = null;
         };
       }, [session, pollInterval, sessionId, authToken]);
       ```

    6. Wire onSeek to VideoInfoPanel — update the existing VideoInfoPanel JSX:
