Implement streamToRTMPS(options: StreamOptions): Promise<void> function using child_process.spawn('ffmpeg', args). Construct args array from research Pattern 2 (lines 139-160 of RESEARCH.md):
- `-re` for native frame rate
- `-stream_loop -1` if options.loop is true
- `-i ${options.videoFile}`
- Video encoding: `-c:v libx264 -b:v 3500k -maxrate 3500k -bufsize 7000k -pix_fmt yuv420p -s 1920x1080 -r 30 -profile:v main -preset veryfast`
- Key frames: `-force_key_frames "expr:gte(t,n_forced*2)" -x264opts "nal-hrd=cbr:no-scenecut"`
- Audio encoding: `-c:a aac -b:a 160k -ar 44100 -ac 2`
- Output: `-f flv ${options.rtmpUrl}`

Return Promise that resolves on ffmpeg process close with code 0, rejects on non-zero code. Pipe stderr to onProgress callback if provided.
