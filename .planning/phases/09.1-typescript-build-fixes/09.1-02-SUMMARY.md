## Changes Made

### backend/src/domain/reaction.ts

Replaced two `export enum` declarations with `const` object + type alias pattern:

**Before:**
```typescript
export enum EmojiType {
  HEART = 'heart',
  FIRE = 'fire',
  CLAP = 'clap',
  LAUGH = 'laugh',
  SURPRISED = 'surprised',
}

export enum ReactionType {
  LIVE = 'live',
  REPLAY = 'replay',
}
```

**After:**
```typescript
export const EmojiType = {
  HEART: 'heart',
  FIRE: 'fire',
  CLAP: 'clap',
  LAUGH: 'laugh',
  SURPRISED: 'surprised',
} as const;
export type EmojiType = typeof EmojiType[keyof typeof EmojiType];

export const ReactionType = {
  LIVE: 'live',
  REPLAY: 'replay',
} as const;
export type ReactionType = typeof ReactionType[keyof typeof ReactionType];
```

### web/src/features/replay/ReplayViewer.tsx

Removed `useMemo` from React import (line 5):

**Before:** `import { useState, useEffect, useMemo } from 'react';`
**After:** `import { useState, useEffect } from 'react';`

### backend/src/handlers/__tests__/stream-started.test.ts

- Removed `mockContext` and `mockCallback` variables (lines 19-20)
- Changed both handler calls from `handler(event, mockContext, mockCallback)` to `handler(event)`
