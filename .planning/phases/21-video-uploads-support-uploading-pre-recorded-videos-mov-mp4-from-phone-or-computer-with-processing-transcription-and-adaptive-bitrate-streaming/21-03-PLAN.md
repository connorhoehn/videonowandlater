# Phase 19 Integration (Transcription Pipeline)

From Phase 19, the recording-ended handler triggers transcription via EventBridge. We'll follow the same pattern:

Phase 19 EventBridge rule pattern:
- Event source: IVS RecordingStateChange
- Event detail type: "IVS Recording State Change"
- Event detail status: ENDED

For Phase 21, we need similar rule:
- Event source: mediaconvert
- Event detail type: "MediaConvert Job State Change"
- Event detail status: SUCCEEDED or FAILED

Phase 19 handler (store-transcript) expects:
```typescript
{
  detail: {
    sessionId: string;  // Encoded in job name or passed in event
    recordingHlsUrl: string;  // From session record
  }
}
```

Phase 21 on-mediaconvert-complete will:
1. Parse MediaConvert job name to extract sessionId
2. Update session recordingHlsUrl = `s3://{bucket}/hls/{sessionId}/master.m3u8` (via CloudFront CDN)
3. Publish EventBridge event to trigger Phase 19 transcription

```typescript

  Name: 'Apple HLS',
  OutputGroupSettings: {
    Type: 'HLS_GROUP_SETTINGS',
    HlsGroupSettings: {
      Destination: `s3://${bucket}/hls/${sessionId}/`,
      SegmentLength: 10,  // 10-second segments
      MaxGopLength: 30,   // 30 frames = ~1 second at 30fps
      // Bitrate ladder: 3 renditions

  Outputs: [

            FrameheightInPixels: 1080,
            FrameWidthInPixels: 1920,

        },
```

```typescript
import { SNSEvent, SNSEventRecord } from 'aws-lambda';
import { MediaConvertClient, CreateJobCommand } from '@aws-sdk/client-mediaconvert';
import { getSession } from '../repositories/session-repository';
import { updateConvertStatus } from '../repositories/session-repository';

interface MediaConvertInput {
  sessionId: string;
  s3Bucket: string;
  s3Key: string;
  sourceFileName: string;
  sourceFileSize: number;
}

export const handler = async (event: SNSEvent): Promise<void> => {
  try {
    const tableName = process.env.TABLE_NAME!;
    const roleArn = process.env.MEDIACONVERT_ROLE_ARN!;
    const outputBucket = process.env.RECORDINGS_BUCKET!;

    for (const record of event.Records) {
      const message = JSON.parse(record.Sns.Message) as MediaConvertInput;
      const { sessionId, s3Bucket, s3Key, sourceFileName } = message;

      console.log(`Starting MediaConvert job for session: ${sessionId}`);

      // Verify session exists
      const session = await getSession(tableName, sessionId);
      if (!session) {
        console.error(`Session not found: ${sessionId}`);
        continue;
      }

      // Generate job name (matches Phase 19 pattern)
      const jobName = `vnl-${sessionId}-${Date.now()}`;

      const mediaConvert = new MediaConvertClient({ region: process.env.AWS_REGION });

      const response = await mediaConvert.send(
        new CreateJobCommand({
          Name: jobName,
          Role: roleArn,
          Input: {
            FileInput: `s3://${s3Bucket}/${s3Key}`,
          },
          OutputGroups: [
            {
              Name: 'Apple HLS',
              OutputGroupSettings: {
                Type: 'HLS_GROUP_SETTINGS',
                HlsGroupSettings: {
                  Destination: `s3://${outputBucket}/hls/${sessionId}/`,
                  SegmentLength: 10,
                  MaxGopLength: 30,
                },
              },
              Outputs: [
                {
                  NameModifier: '1080p',
                  VideoDescription: {
                    CodecSettings: {
                      H264Settings: {
                        MaxBitrate: 5000000,  // 5 Mbps
                        FrameHeight: 1080,
                        FrameWidth: 1920,
                      },
                    },
                  },
                },
                {
                  NameModifier: '720p',
                  VideoDescription: {
                    CodecSettings: {
                      H264Settings: {
                        MaxBitrate: 2500000,  // 2.5 Mbps
                        FrameHeight: 720,
                        FrameWidth: 1280,
                      },
                    },
                  },
                },
                {
                  NameModifier: '480p',
                  VideoDescription: {
                    CodecSettings: {
                      H264Settings: {
                        MaxBitrate: 1200000,  // 1.2 Mbps
                        FrameHeight: 480,
                        FrameWidth: 854,
                      },
                    },
                  },
                },
              ],
            },
          ],
        })
      );

      const jobId = response.Job?.Id!;
      console.log(`MediaConvert job submitted: ${jobName} (ID: ${jobId})`);

      // Update session with job name and pending status
      await updateConvertStatus(tableName, sessionId, jobName, 'pending');
    }
  } catch (error) {
    console.error('start-mediaconvert error:', error);
    // Don't rethrow; SNS message is consumed even if handler fails
  }
};
```

Key points:
- SNSEvent has Records[]. Parse each Record.Sns.Message as JSON
- jobName format matches Phase 19: vnl-{sessionId}-{epochMs}
- H.264 codec ensures browser compatibility (H.265 input is re-encoded)
- 3 renditions cover typical bandwidth scenarios (mobile, tablet, desktop)
- Output destination: s3://{bucket}/hls/{sessionId}/ (matches Phase 21 expected path)
- Non-blocking error handling: continue on individual session failures

```typescript
import { EventBridgeEvent } from 'aws-lambda';
import { EventBridgeClient, PutEventsCommand } from '@aws-sdk/client-eventbridge';
import { getSession, updateSessionRecording } from '../repositories/session-repository';

interface MediaConvertJobDetail {
  jobName: string;
  jobId: string;
  status: 'SUBMITTED' | 'PROGRESSING' | 'COMPLETE' | 'CANCELED' | 'ERROR';
  outputGroupDetails?: Array<{
    playlistFile?: string;  // e.g., "master.m3u8"
  }>;
}

export const handler = async (
  event: EventBridgeEvent<'MediaConvert Job State Change', MediaConvertJobDetail>
): Promise<void> => {
  try {
    const tableName = process.env.TABLE_NAME!;
    const bucket = process.env.RECORDINGS_BUCKET!;
    const eventBusName = process.env.EVENT_BUS_NAME!;

    const detail = event.detail;
    const { jobName, jobId, status } = detail;

    console.log(`MediaConvert job state change: ${jobName} (${jobId}) → ${status}`);

    // Parse sessionId from jobName (format: vnl-{sessionId}-{epochMs})
    const jobNameMatch = jobName.match(/^vnl-([a-z0-9-]+)-\d+$/);
    if (!jobNameMatch) {
      console.error(`Could not parse sessionId from jobName: ${jobName}`);
      return;
    }
    const sessionId = jobNameMatch[1];

    // Get session
    const session = await getSession(tableName, sessionId);
    if (!session) {
      console.error(`Session not found: ${sessionId}`);
      return;
    }

    if (status === 'COMPLETE') {
      // MediaConvert job succeeded
      const recordingHlsUrl = `s3://${bucket}/hls/${sessionId}/master.m3u8`;

      console.log(`Updating session with HLS URL: ${recordingHlsUrl}`);

      // Update session with recording metadata
      await updateSessionRecording(tableName, sessionId, {
        recordingHlsUrl,
        recordingStatus: 'available',
        convertStatus: 'available',
        status: 'ended',
        // recordingDuration can be fetched from MediaConvert job details if available
      });

      // Publish event to trigger Phase 19 transcription pipeline
      const eventBridge = new EventBridgeClient({ region: process.env.AWS_REGION });
      await eventBridge.send(
        new PutEventsCommand({
          Entries: [
            {
              Source: 'vnl.upload',
              DetailType: 'Upload Recording Available',
              Detail: JSON.stringify({
                sessionId,
                recordingHlsUrl,
              }),
              EventBusName: eventBusName,
            },
          ],
        })
      );

      console.log(`Transcription pipeline triggered for session: ${sessionId}`);
    } else if (status === 'ERROR' || status === 'CANCELED') {
      // MediaConvert job failed
      console.error(`MediaConvert job failed: ${jobName} (${jobId})`);

      // Mark session as failed
      await updateSessionRecording(tableName, sessionId, {
        convertStatus: 'failed',
        uploadStatus: 'failed',
      });
    }
  } catch (error) {
    console.error('on-mediaconvert-complete error:', error);
    // Don't rethrow; EventBridge message is consumed even if handler fails
  }
};
```

Key points:
- Parse sessionId from jobName using regex: /^vnl-([a-z0-9-]+)-\d+$/
- recordingHlsUrl = s3://{bucket}/hls/{sessionId}/master.m3u8 (matches MediaConvert output structure)
- Update session status='ended' ONLY after recordingHlsUrl is set (avoids "session status confusion" pitfall)
- EventBridge event trigger for Phase 19 transcription uses detail: { sessionId, recordingHlsUrl }
- Non-blocking error handling: log errors but don't rethrow
- Handle both ERROR and CANCELED status codes (both indicate job failure)

Note: updateSessionRecording() is a new function (or enhance existing updateSession()) that atomically updates multiple recording fields. See repository task below.

```typescript
/**
 * Update session with recording metadata after MediaConvert completion
 * Atomic update ensures recordingHlsUrl, recordingStatus, and status all update together
 */
export async function updateSessionRecording(
  tableName: string,
  sessionId: string,
  updates: {
    recordingHlsUrl?: string;
    recordingStatus?: string;
    recordingDuration?: number;
    convertStatus?: string;
    uploadStatus?: string;
    status?: string;
  }
): Promise<void> {
  const docClient = getDocumentClient();

  // Build UpdateExpression dynamically based on provided fields
  const fields: string[] = [];
  const values: Record<string, any> = {};

  if (updates.recordingHlsUrl) {
    fields.push('recordingHlsUrl = :hlsUrl');
    values[':hlsUrl'] = updates.recordingHlsUrl;
  }
  if (updates.recordingStatus) {
    fields.push('recordingStatus = :recordingStatus');
    values[':recordingStatus'] = updates.recordingStatus;
  }
  if (updates.recordingDuration) {
    fields.push('recordingDuration = :duration');
    values[':duration'] = updates.recordingDuration;
  }
  if (updates.convertStatus) {
    fields.push('convertStatus = :convertStatus');
    values[':convertStatus'] = updates.convertStatus;
  }
  if (updates.uploadStatus) {
    fields.push('uploadStatus = :uploadStatus');
    values[':uploadStatus'] = updates.uploadStatus;
  }
  if (updates.status) {
    fields.push('status = :status');
    values[':status'] = updates.status;
  }

  if (fields.length === 0) {
    return; // No updates provided
  }

  await docClient.send(new UpdateCommand({
    TableName: tableName,
    Key: {
      PK: `SESSION#${sessionId}`,
      SK: 'METADATA',
    },
    UpdateExpression: `SET ${fields.join(', ')}`,
    ExpressionAttributeValues: values,
  }));
}
```

This function allows on-mediaconvert-complete to atomically update recording fields without separate calls. It follows the field-isolation pattern established in Plan 21-01.
