Handlers are already wired for POST /sessions:

```typescript
const createSessionFunction = new NodejsFunction(this, 'CreateSessionFunction', {
  runtime: Runtime.NODEJS_20_X,
  handler: 'handler',
  entry: path.join(__dirname, '../../backend/src/handlers/create-session.ts'),
  timeout: Duration.seconds(30),
  environment: {
    TABLE_NAME: sessionTable.tableName,
    CHANNEL_POOL_NAME: channelPoolName,
  },
});

sessionTable.grantReadWriteData(createSessionFunction);
recordingsBucket.grantReadWrite(createSessionFunction);

// API Gateway integration
const sessionsResource = api.root.addResource('sessions');
sessionsResource.addMethod(
  'POST',
  new LambdaIntegration(createSessionFunction),
  {
    authorizer: cognitoAuthorizer,
    authorizationType: AuthorizationType.COGNITO,
  }
);
```

  runtime: Runtime.NODEJS_20_X,

  timeout: Duration.seconds(30),
  environment: {
    TABLE_NAME: sessionTable.tableName,
    RECORDINGS_BUCKET: recordingsBucket.bucketName,
  },
});

const completeUploadFunction = new NodejsFunction(this, 'CompleteUploadFunction', {
  runtime: Runtime.NODEJS_20_X,
  handler: 'handler',
  entry: path.join(__dirname, '../../backend/src/handlers/complete-upload.ts'),

    TABLE_NAME: sessionTable.tableName,

2. Grant permissions to handlers:

```typescript
// Grant DynamoDB access to all three functions
sessionTable.grantReadWriteData(initUploadFunction);
sessionTable.grantReadWriteData(getPartPresignedUrlFunction);

// Grant S3 access to all three functions
recordingsBucket.grantReadWrite(initUploadFunction);
recordingsBucket.grantReadWrite(getPartPresignedUrlFunction);
recordingsBucket.grantReadWrite(completeUploadFunction);

// Grant SNS publish to complete-upload function
mediaConvertTopic.grantPublish(completeUploadFunction);
```

Note: `mediaConvertTopic` is created in session-stack.ts. Import it via stack props or use StackUtils pattern (check existing code for reference).
