API Endpoints (created by Plan 01):
```
GET /sessions/live
  Auth: Cognito JWT (required)
  Response: { sessions: Session[] }
  Sessions have: sessionId, userId, sessionType, status, createdAt

PUT /sessions/{sessionId}/spotlight
  Auth: Cognito JWT (required)
  Body: { featuredCreatorId: string | null, featuredCreatorName: string | null }
  Response: { message: string, featuredCreatorId, featuredCreatorName }
  Validates: session ownership, privacy guards

GET /sessions/{sessionId}
  Auth: Cognito JWT (required)
  Response: Session object (includes featuredCreatorId, featuredCreatorName)
```

Session type (extended in Plan 01):
```typescript
interface Session {
  sessionId: string;
  userId: string;
  sessionType: 'BROADCAST' | 'HANGOUT' | 'UPLOAD';
  status: 'creating' | 'live' | 'ending' | 'ended';
  createdAt: string;
  isPrivate?: boolean;
  featuredCreatorId?: string;    // NEW: sessionId of featured creator
  featuredCreatorName?: string;  // NEW: username of featured creator
  // ... other fields
}
```
