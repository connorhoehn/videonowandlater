AWS Transcribe job name constraints: max 200 chars, must match [0-9a-zA-Z._-]+
MediaConvert jobId format: typically `<13-digit-epoch>-<6-hex>` (e.g., `1741723938123-abc123`) — ~20 chars.
`vnl-${sessionId}-${jobId}` = ~60 chars total. Safe.
