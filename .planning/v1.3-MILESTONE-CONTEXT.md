# v1.3 Milestone Context

**Milestone:** v1.3 Secure Sharing
**Date:** 2026-03-06

## User Input

**Focus areas for v1.3:**
1. ✅ Shareable session links — Generate time-limited shareable links for broadcasts/recordings without requiring viewers to create accounts
2. ✅ Private collections — Users organize recordings into private/public playlists or collections for easy sharing

**Out of scope (for v1.3):**
- Permission-based access (granular permissions) — Deferred to v2 as todo item

**Key constraint:** "Don't get into users" — Stay focused on sharing mechanics, not user profiles/management.

## Vision for v1.3

Enable users to securely share their recordings and live sessions through shareable links and organized collections, making it easy to curate and distribute content without building user management complexity.
